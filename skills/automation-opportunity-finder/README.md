# automation-opportunity-finder

Helps professionals identify practical AI automation Quick Wins through guided discovery, workflow analysis, role-based suggestions and industry-specific patterns. Covers 13 industries and 6 role types. Designed for people who do not know where to start with AI automation.

---

## How to install

Copy this folder into your Claude Code skills directory:

```
skills/automation-opportunity-finder/  →  ~/.claude/skills/automation-opportunity-finder/
```

Then use it by typing `/automation-opportunity-finder` in Claude Code.

---

## How to use

Run `/automation-opportunity-finder` and the skill will guide you through:

1. **Discovery** — questions about your role, industry, and weekly tasks
2. **Assessment** — scoring each opportunity against 7 criteria
3. **Recommendation** — top 3 Quick Wins with time savings, tooling, and next steps

Or say **"just suggest something"** to skip straight to a universal Quick Win you can build in under an hour.
