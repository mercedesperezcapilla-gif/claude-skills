# ai-use-case-business-case-builder

Turns a scoped AI use case into a one-page internal business case that wins buy-in. Returns the problem and its current cost, the proposed automation in plain English, a transparent ROI calculation, governance considerations, and a clear ask. Pairs with the FS Governance Tier Checker — scope the tier, then build the case to fund it.

---

## How to install

Copy this folder into your Claude Code skills directory:

```
skills/ai-use-case-business-case-builder/  →  ~/.claude/skills/ai-use-case-business-case-builder/
```

Then use it by typing `/ai-use-case-business-case-builder` in Claude Code.

---

## How to use

Describe the AI use case you want to fund, and the skill will return a one-page case:

1. **The problem** — current process and what it costs
2. **The proposed automation** — plain English, with what stays human
3. **The numbers** — ROI calculation, shown transparently
4. **Governance considerations** — proportionate to the tier
5. **The ask** — exactly what you need to proceed

Have your frequency, time-per-cycle and who-does-it ready, or the skill will help you estimate.
